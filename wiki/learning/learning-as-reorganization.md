# Learning as Reorganization

Learning is not the accumulation of information but the process by which a complex system -- organism, group, organization -- modifies its attractors through interaction with its environment. The storage metaphor (input-processing-storage) predicts poorly: knowing a lot *about* something does not mean you can *do* it, and conditions that boost immediate performance often degrade long-term learning. Because performance and learning are distinct phenomena, and because the system that learns is not the brain alone but the entire organism in interaction with its environment, learning must be understood as structural reorganization rather than information transfer.

## Key points

- **The storage metaphor fails for complex skills.** If learning were accumulation, then knowing three books about negotiation should make you a good negotiator. But knowing *about* negotiation and being able to negotiate are different things -- the former is information, the latter is a reorganized system of perception, action, and judgment. The model of learning-as-accumulation fails precisely where complex skills are concerned.
- **Performance and learning are distinct phenomena.** Blocked repetition and constant correction improve short-term performance but degrade long-term learning. Varied practice and desirable difficulties worsen immediate performance but produce more robust, transferable learning. This paradox only makes sense if learning is reorganization: identical repetition consolidates existing attractors, while variation generates new ones.
- **Error is not failure but a reorganization signal.** Information alone does not change a system. Identical repetition consolidates existing attractors while variation generates new ones. Error signals that the system is being sufficiently perturbed to reorganize -- it is the mechanism, not an obstacle to be eliminated.

### Repetition without repetition

- **Bernstein's principle: repeat the problem, not the solution.** Each repetition should generate a context-adapted solution rather than recording a fixed pattern. The system is not memorizing a solution but extracting the principles that generate solutions, which requires variability to abstract those principles.
- **Practicing in one context couples the pattern too tightly.** If the system only practices in one scenario, the emergent pattern becomes specific to those conditions and fragile when transferred. Variability during practice is exploration in the exploration-vs-exploitation sense; without it, the system only exploits what it already has.
- **This reframes what counts as a "good practice session."** Not one where everything goes well, but one where the system is perturbed enough to need reorganization yet capable enough to find a new solution.
- **The body has too many degrees of freedom for centralized control.** Bernstein showed that movement problems are solved through self-organization, not by executing motor programs. The body, nervous system, and environment form a self-organizing unit where solutions emerge from interaction.

### Expert perception as emergent property

- **De Groot and Chase-Simon's chess experiments reveal reorganized perception.** Grandmasters recall real game positions vastly better than novices, but the advantage disappears with random positions. The expert's perceptual system has reorganized to see relationships, tensions, and action possibilities -- not individual pieces. Asking an expert reader to recall random letter sequences produces the same result: the advantage disappears because there are no meaningful structures to perceive.
- **Expert intuition is distributed pattern recognition.** It is distributed across the entire system (body, nervous system, perception patterns), not stored as explicit rules. It works well under pressure because it bypasses the bottleneck of consciousness, but fails in unfamiliar domains because the system has no attractors there.
- **Perception extends beyond vision.** An expert in linear algebra perceives structure where a novice sees noise, just as an adult reads words where a child sees scribbles. The difference lies not in what they know but in how the whole system has reorganized through years of interaction. Mastery transforms perception itself.

### Embodied anticipation

- **Internal simulation is grounded in the body.** The nervous system enables internal simulation of possible actions without executing them -- a bodily rehearsal where the brain partially activates motor and sensory circuits based on prior experience. This is not abstract thinking but embodied simulation.
- **Expert anticipation extends this capacity.** An experienced chess player does not just see more patterns but can mentally "play" several moves ahead because the system has internalized thousands of games. The simulation is fast, automatic, and embodied.
- **Anticipation has limits.** If prior experience is limited, simulations will be poor. If the system has very strong attractors, it will tend to simulate only the solutions it already knows. Anticipation is powerful but as conditioned by history as perception is.

### The extended learning system

- **Gibson's ecological perception.** Affordances are not in the world or in the body but in the relationship between them, so perception and action are inseparable. Learning a skill is learning to perceive-and-act as a single unit.
- **Clark's extended cognition.** The cognitive system does not end at the skin. Tools integrate into cognition and transform it. Because the medium changes the thought, the learning system is organism + body + environment.

## Examples

- **Knowing about vs. knowing how.** Knowing three books about negotiation yet freezing in the first real negotiation. The information is there, but the system has not reorganized to perceive and act in the pressured, dynamic situation.
- **Blocked vs. varied practice.** Practicing always in block boosts immediate performance but produces fragile learning tied to specific conditions. Varied practice feels worse in the moment but produces durable learning because the system extracts general principles.
- **Chess expertise.** Grandmasters see open diagonals, weak formations, and latent threats at a glance without conscious analysis. With random piece positions, the advantage disappears -- because there are no meaningful structures to perceive.
- **Jumping a puddle.** The body "calculates" whether it can clear the distance before you jump -- it simulates the jump based on what the body has done in similar situations, and the decision to jump or walk around is often made before conscious awareness. This is embodied anticipation in action.
- **Learning to write by hand.** Hand, fingers, forearm muscles, spinal motor neurons, and brain reorganize together -- not as a "download" of instructions. The learning is distributed across the entire body-environment system.
- **Calculating with different tools.** Head vs. fingers vs. pencil-and-paper vs. calculator: in every case *we* calculate, but each tool changes what we can do and therefore how we think. The medium changes the thought.
- **Negotiation variability.** Always practicing the same scenario builds a solution fitted only to that case. Varied contexts -- aggressive counterparts, different cultures, phone negotiations -- let the system extract general principles rather than memorizing a single pattern.
