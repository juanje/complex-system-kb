# Agent

**Domain:** Complex systems, agent-based modeling, organizational theory

An agent is an autonomous entity with the capacity to perceive its environment, process information, and make decisions based on that perception. Agents do not need to be conscious or intelligent — cells in the body act as agents following simple rules without awareness, atoms "sense" other atoms through electromagnetic forces, and banking entities detect changes through financial data analysis. What makes something an agent is the perceive-decide-act loop, not the sophistication of the decision-making.

## Usage context
- In complexity science, agents are the fundamental building blocks of self-organizing systems. Every complex system — from ecosystems to organizations to the human body — is composed of agents interacting locally according to their own rules.
- Agents always react based on perception (changes in internal state or external environment), never randomly. There is always a lag between what is perceived and the reaction, which has important implications for system dynamics.
- Perception is unique to each agent: no external observer or other agent can perceive and interact with the environment in the same way, which means external perspectives can never provide the information an agent needs to react appropriately.
- Agents exist at multiple levels of a system: cells are agents of the body, individuals are agents in teams, teams are agents in organizations — connecting the concept to the multilevel nature of complex systems.
- The distinction between [strong and weak agents](strong-and-weak-agents.md) determines how predictable the system is: weak agents follow fixed rules, strong agents learn and create new rules.
