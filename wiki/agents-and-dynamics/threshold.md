# Threshold

**Domain:** Complex systems, agent dynamics, phase transitions

A threshold is a critical value that, once exceeded, triggers a change in the state or behavior of a system or agent. Thresholds represent tipping points: below the threshold, the system remains in its current state; above it, the system transitions to a qualitatively different state. Thresholds exist at both the agent level (an individual agent crossing its threshold changes its own behavior) and the system level (enough agents crossing their thresholds can trigger a system-wide change).

## Usage context
- When a threshold is crossed, the system either transitions to a new stable state (phase change) or reaches saturation where its capacity degrades, potentially leading to collapse. The sponge analogy captures this: a sponge absorbs liquid until it reaches its threshold, at which point it makes things wet instead of drying them.
- Diversity of thresholds among agents can dramatically accelerate system change. Because agents with lower thresholds trigger first, their actions push others past their thresholds, creating cascading avalanches — a mechanism central to the Schelling segregation model and to understanding how peaceful demonstrations can escalate into violence.
- Thresholds are closely related to meta-rules in strong agents: crossing a threshold can trigger a change in which meta-rules apply, altering the agent's behavior pattern entirely.
- In biological systems, thresholds govern phase transitions like water changing state, fever responses activating at specific temperatures, and stress hormones triggering fight-or-flight at specific intensity levels.
